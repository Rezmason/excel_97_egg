### About
This is a JS port of the flight simulator [hidden](https://eeggs.com/items/718.html) within [Microsoft Excel 97](https://en.wikipedia.org/wiki/Microsoft_Excel#Excel_97_(v8.0)). It's built on top of [three.js](https://threejs.org).

This README file is a work in progress.
